# Как выбрать цвет
На [веб-сервиса](https://yzoop.github.io/) реализовано 4 выбора цвета: 1 графический и 3 цифровые.
## Графический выбор цвета
На той же странице для выбора цвета графическим способом есть цветовая палитра, которая выглядит следующим образом:
![alt text](https://i.ibb.co/jTFcZnT/screen2.png)
Чтобы выбрать цвет, нужно выбрать оттенок на левой палитре,а затем на главном прямоугольнике выбрать оттенок. На выбранный оттенок указывает кружочек.
## Цифровой выбор цвета
### HEX
Поле для выбора цвета находится ниже графического блока в белом прямоугольнике. Чтобы ввести цвет в формате HEX, нужно нажать на поле для ввода и ввести собственное значение, предварительно удалив частично или полностью текущее значение
https://i.ibb.co/PwSRZYM/screen3.png
### RGB
Так же в белом блоке есть 3 отдела для ввода R, G и B значений каналов.
![](https://i.ibb.co/bKkhqqh/screen4.png)
### HSV
То же самое для ввода цвета в формате HSV <br>
![](https://i.ibb.co/cxfXssY/screen5.png)
